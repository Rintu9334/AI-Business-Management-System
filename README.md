# AI Business Management System

## Description
Role-based system where:
- Admin manages users & system
- Manager manages projects & tasks
- Employee completes tasks

## ⚙️ Technologies
- Django (Backend)
- React (Frontend)
- SQLite (Database)

## 🔥 Features
- Authentication system
- Task management
- Attendance tracking
- Performance analysis
- AI predictions

## 🎯 Purpose
This project is developed for academic submission and demonstration.
